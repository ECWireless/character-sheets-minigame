# character-sheets-minigame

A MUD-based minigame for CharacterSheets

## Rules

Read about the initial concept of the game [here](https://hackmd.io/@ecwireless/r1TznH08a).

## Setup

1. Ensure you have `node`, `pnpm` and `foundry` installed.

2. Install packages

```bash
pnpm install
```

3. Run server

```bash
pnpm dev
```
